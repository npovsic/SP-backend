Requisites:
	- NodeJS (https://nodejs.org/en/download/)
	- MongoDB (https://www.mongodb.com/download-center?jmp=nav)

Process:
	- set environmental variables in Windows
	- open two terminal windows in project root
		- first:
			> npm run db
		- second:
			> npm start
	- open a browser window:
		- url: localhost:3000
